CSS Premenné Úloha: Vytvor premenné v :root a použi ich v kartách a tlačidlách Výhody CSS premenných Premenné umožňujú uložiť hodnoty (farby, veľkosti, medzery) a znovu ich použiť. Ak chcete zmeniť farbu na celej stránke, zmeníte ju len na jednom mieste! Primárne tlačidlo Sekundárne tlačidlo 2. Selektory - Vyberanie elementov A) Selektor elementu - Všetky odstavce Úloha: Nastav, aby VŠETKY odstavce mali veľkosť písma 18px, farbu #2c3e50 a riadkovanie 1.6 Tento odstavec má štýl aplikovaný cez selektor elementu. Aj tento odstavec má rovnaký štýl. Všetky odstavce budú vyzerať rovnako. B) Selektor triedy - Len vybrané elementy Úloha: Nastav, aby IBA elementy s triedou dolezite mali červenú farbu, tučné písmo, ružové pozadie a vnútorný okraj 10px Tento text je označený ako dôležitý pomocou triedy dolezite Tento text nie je dôležitý - nemá triedu. Aj tento text je dôležitý - má triedu dolezite C) Selektor ID - Jedinečný element Úloha: Nastav, aby IBA element s ID hlavna-poznamka mal žlté pozadie, tmavý text, žlté orámovanie a vnútorný okraj 15px Toto je hlavná poznámka s jedinečným ID hlavna-poznamka Tento text je obyčajný odstavec bez ID. 3. Box Model - Vnútorný okraj, Orámovanie, Vonkajší okraj A) Box s vnútorným okrajom Úloha: Vytvor box s vnútorným okrajom 30px, modrým orámovaním a svetlým pozadím Box s vnútorným okrajom 30px Všimnite si priestor medzi týmto textom a modrým orámovaním. To je vnútorný okraj elementu. B) Box s asymetrickým vnútorným okrajom Úloha: Vytvor box, kde vnútorný okraj je rôzny na každej strane (10px zhora, 20px zprava, 30px zdola, 40px zľava) Box s asymetrickým vnútorným okrajom Vnútorný okraj: 10px zhora, 20px zprava, 30px zdola, 40px zľava C) Box s veľkým vonkajším okrajom Úloha: Vytvor box s vonkajším okrajom 50px (veľký priestor okolo boxu) Box s veľkým vonkajším okrajom 50px Všimnite si veľký priestor OKOLO tohto boxu. To je vonkajší okraj elementu. ✅ Čo sme sa naučili? 1. CSS Premenné Definícia: :root { --nazov: hodnota; } Použitie: color: var(--nazov); Výhoda: Zmena na jednom mieste = zmena všade! 2.
