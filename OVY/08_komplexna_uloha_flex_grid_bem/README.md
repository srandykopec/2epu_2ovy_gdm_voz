# 🚀 Mini Media Queries Cvičenie

Responzívna stránka pre precvičenie **Flexbox, Grid a Media Queries**.

## 📁 Štruktúra

```

starter/           
├── index.html      
├── README.md 
└── css/
    ├── normalize.css
    └── style.css   

```

## 🎯 Čo budete vytvárať

**Responzívnu cestovnú kanceláriu** s:
- Header + navigácia
- Hero sekcia
- 3 karty výhod
- 3 destinácie
- Footer s kontaktom

## 📱 Breakpointy

- **Mobile**: 0-767px (predvolené)
- **Tablet**: 768px+ 
- **Desktop**: 1024px+

## 🛠️ Technológie

### **Flexbox použite pre:**
- Header (logo + nav vedľa seba na tablet)
- Navigácia (column → row)

### **Grid použite pre:**
- **Výhody**: 
  - Mobile: 1 stĺpec
  - Tablet: 3 stĺpce
- **Destinácie**: 
  - Mobile: 1 stĺpec
  - Tablet: 2 stĺpce
  - Desktop: 3 stĺpce
- **Footer**: 
  - Mobile: 1 stĺpec
  - Tablet: 2 stĺpce
